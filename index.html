Tactical Tower Defense

A very small project where I experiment in a 3D plane and learn using JSON files, the level layouts and enemy waves are all configured using a JSON file and a JSON reader, the premise of the game is the player placing down units (in this case a knight) to block enemies in their paths before they reach the exit, certain units have a certain amount of block so unless you beat them quick you'll need more, other units can also be placed on the slightly lowered highground such as archers, unfortunately I haven't yet implemented these nor do I have a video to view for this project.
